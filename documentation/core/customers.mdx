---
title: Customers
---

The Modem Pay API’s customer management is built to help you store and manage customer details efficiently. Below are detailed steps and example requests for creating, retrieving, updating, deleting, and listing customers, as well as listing their payment methods. This allows you to manage customer profiles and payment preferences seamlessly.

---

#### **Create a Customer**

To create a customer, use the `modem.customers.create()` method, which allows you to add key information like the customer's name, email, phone number, and an optional starting balance. The `distinct` option, when set to `true`, checks for existing customers with matching fields (e.g., email) to avoid duplicate entries. If `distinct` is omitted or set to `false`, a new customer entry will be created without duplicate checks.

**Sample Request:**

<CodeGroup>
```typescript nodejs
const customer = await modem.customers.create(
    { name: "Caleb Okpara", email: "caleb@modempay.com", phone: "7000001" },
    { distinct: true }
);
```
</CodeGroup>

#### **Retrieve a Customer**

To fetch a customer’s details, use `modem.customers.retrieve(customerId)`. Replace `customerId` with the customer’s unique ID.

**Sample Request:**

<CodeGroup>
```javascript nodejs 
const customer = await modem.customers.retrieve("8a8c8a0e-c5d2-4692-a94b-25284d36adc7"); 
```
</CodeGroup>

**Sample Response:**

  "name": "Caleb Okpara",

#### **Update a Customer**

Update a customer's information using `modem.customers.update(customerId, params)`. Specify the customer ID and the fields to update.

**Sample Request:**

<CodeGroup>
```javascript nodejs 
const customer = await modem.customers.update(
    "8a8c8a0e-c5d2-4692-a94b-25284d36adc7",
    { name: "Chibuike Okpara" }
);
```
</CodeGroup>

#### **Delete a Customer**

Remove a customer from your records using `modem.customers.delete(customerId)`. This action is irreversible.

**Sample Request:**

<CodeGroup>
```javascript nodejs 
await modem.customers.delete("8a8c8a0e-c5d2-4692-a94b-25284d36adc7");
```
</CodeGroup>

---

#### **List All Customers**

To view all customers, use `modem.customers.list()`. This returns a list of customers, with pagination options if there are many entries.

**Sample Request:**

<CodeGroup>
```javascript nodejs 
const customers = await modem.customers.list({ limit: 2 });

// or list by matching terms
const customers = await modem.customers.list({ limit:2, search: "caleb" })

```
</CodeGroup>
